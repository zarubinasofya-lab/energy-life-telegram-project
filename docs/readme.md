\# Energy Life Telegram Project



\## Описание проекта



\*\*Energy Life Telegram Project\*\* — это Telegram-экосистема для канала «Энергия жизни🐦‍🔥».



Проект включает Telegram-канал и двух ботов:



1\. \*\*@energy\_life\_content\_bot\*\* — админ-бот для генерации, подготовки и автопубликации контента.

2\. \*\*@energy\_life\_fohow\_bot\*\* — пользовательский бот для знакомства с проектом, прохождения теста и сбора лидов.



\## Цель проекта



Автоматизировать ведение Telegram-канала, упростить публикацию контента и организовать сбор заявок от заинтересованных пользователей.



\## Функциональность



\### Контент-бот



\* генерация постов по рубрикам;

\* ручная публикация постов;

\* автопостинг по расписанию;

\* публикация опросов;

\* автоматическое добавление хэштегов;

\* подбор случайных картинок;

\* публикация в Telegram-канал.



\### Тест-бот



\* приветственное меню;

\* описание проекта;

\* тест из 8 вопросов;

\* определение типа пользователя;

\* категории: Клиент, Партнер, Лидер;

\* уведомление администратора;

\* сохранение данных в Google Sheets.



\## Технологии



\* Python;

\* python-telegram-bot;

\* OpenAI API;

\* APScheduler;

\* Google Sheets API;

\* gspread;

\* Telegram Bot API.



\## Результат



В результате создана рабочая система для Telegram-канала:



\* канал получает регулярный контент;

\* администратор может публиковать посты вручную;

\* подписчики могут проходить тест;

\* лиды автоматически приходят админу и сохраняются в Google-таблицу;

\* проект оформлен как кейс для портфолио.



\## Безопасность



В репозиторий не добавляются:



\* реальные токены Telegram-ботов;

\* OpenAI API-ключ;

\* credentials.json;

\* config.py с настоящими данными.



Для примера используется файл config.example.py.
